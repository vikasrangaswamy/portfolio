LIFO is the key insight. Monotonic stacks maintain an order (ascending or descending) and pop when the current element violates that order. Always store indices, not values, so you can refer back to original data. Use stacks for matching brackets, evaluating expressions, and finding next/previous greater/smaller.

## Representative approach

```python
def next_greater(nums):
    """Find next greater element for each position."""
    result = [-1] * len(nums)
    stack = []  # indices
    for i, n in enumerate(nums):
        while stack and nums[stack[-1]] < n:
            result[stack.pop()] = n
        stack.append(i)
    return result

def is_valid_parentheses(s):
    """Check if parentheses/brackets/braces are balanced."""
    stack = []
    pairs = {'(': ')', '[': ']', '{': '}'}
    for char in s:
        if char in pairs:
            stack.append(char)
        else:
            if not stack or pairs[stack.pop()] != char:
                return False
    return len(stack) == 0
```
