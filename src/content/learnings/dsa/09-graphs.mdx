Graph problems require choosing the right representation (adjacency list for sparse, matrix for dense) and the right algorithm. BFS finds shortest path in unweighted graphs; DFS handles reachability and cycles; union-find clusters components.

## Key algorithms

| Algorithm | Data structure | Use case |
|-----------|----------------|----------|
| BFS | Queue (deque) | Shortest path (unweighted), level order |
| DFS | Stack / Recursion | Connected components, path exists |
| Topological Sort | Kahn's (queue) or DFS | Course schedule, build order |
| Union Find | parent[] array | Connected components, detect cycle |
| Dijkstra | Min-heap | Shortest path (weighted) |

## Representative approach

```python
def bfs_shortest_path(graph, start, end):
    """BFS to find shortest path in unweighted graph."""
    from collections import deque
    queue = deque([(start, [start])])
    visited = {start}
    while queue:
        node, path = queue.popleft()
        if node == end:
            return path
        for neighbor in graph.get(node, []):
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append((neighbor, path + [neighbor]))
    return []

def dfs_connected_components(graph):
    """Count connected components using DFS."""
    visited = set()
    count = 0
    def dfs(node):
        visited.add(node)
        for neighbor in graph.get(node, []):
            if neighbor not in visited:
                dfs(neighbor)
    for node in graph:
        if node not in visited:
            dfs(node)
            count += 1
    return count
```
