Binary search works on sorted arrays and monotonic conditions. Choose `while left <= right` for "find exact" and `while left < right` for "find first/last." Watch your mid calculation to avoid overflow.

## Representative approach

```python
def binary_search(nums, target):
    """Find exact position of target in sorted array."""
    lo, hi = 0, len(nums) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1

def find_first(nums, target):
    """Find first occurrence of target in sorted array."""
    lo, hi = 0, len(nums) - 1
    result = -1
    while lo <= hi:
        mid = (lo + hi) // 2
        if nums[mid] == target:
            result = mid
            hi = mid - 1
        elif nums[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return result
```
