Two pointers turn O(n²) problems into O(n) by leveraging sorted order or exploiting symmetric properties. Master opposite ends (for pairs) and same direction (for partitioning).

## Key patterns

| Variant | Example | When |
|---------|---------|------|
| Opposite ends | Two Sum II, Container With Most Water | Sorted array, target sum |
| Same direction (slow/fast) | Remove Duplicates, Move Zeros | In-place partition |
| Three pointers | Three Sum | Fix one, opposite ends on rest |
| Fast & slow (Floyd) | Linked List Cycle | Cycle detection |

## Representative approach

```python
def check_if_palindrome(string):
    """Check palindrome with two pointers from both ends."""
    left, right = 0, len(string) - 1
    while left < right:
        if string[left] != string[right]:
            return False
        left += 1
        right -= 1
    return True

def remove_duplicates(nums):
    """Remove duplicates in-place; return new length."""
    if not nums:
        return 0
    slow = 0
    for fast in range(1, len(nums)):
        if nums[fast] != nums[slow]:
            slow += 1
            nums[slow] = nums[fast]
    return slow + 1
```
