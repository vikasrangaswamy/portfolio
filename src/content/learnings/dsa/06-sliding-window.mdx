Fixed-size windows are straightforward: compute first, then slide by adding right and dropping left. Variable-size windows expand right until invalid, then shrink left until valid again — always use a `while` loop for shrinking, never just `if`.

## Representative approach

```python
def max_subarray_k(nums, k):
    """Find max sum of contiguous subarray of size k."""
    window = sum(nums[:k])
    best = window
    for i in range(k, len(nums)):
        window += nums[i] - nums[i - k]
        best = max(best, window)
    return best

def longest_substring_unique(string):
    """Find longest substring with all unique characters."""
    left = 0
    char_index = {}
    max_len = 0
    for right, char in enumerate(string):
        if char in char_index and char_index[char] >= left:
            left = char_index[char] + 1
        char_index[char] = right
        max_len = max(max_len, right - left + 1)
    return max_len
```
