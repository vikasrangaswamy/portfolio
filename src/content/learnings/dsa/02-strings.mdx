Strings are immutable in Python, so build with a list then `''.join()` for many concatenations. Master frequency counters and two-pointer techniques to solve most string problems efficiently.

## Key techniques

| Technique | Key idea | Time |
|-----------|----------|------|
| Anagram check | Sort both or use frequency counter | O(n log n) or O(n) |
| Palindrome check | Two pointers from ends | O(n) |
| Longest palindromic substring | Expand around center | O(n²) |
| Group anagrams | Hash sorted string as key | O(n·k log k) |

## Representative approach

```python
def is_palindrome(s):
    """Check if string is palindrome with two pointers."""
    left, right = 0, len(s) - 1
    while left < right:
        if s[left] != s[right]:
            return False
        left += 1
        right -= 1
    return True

def group_anagrams(words):
    """Group words that are anagrams of each other."""
    groups = {}
    for word in words:
        key = ''.join(sorted(word))
        groups.setdefault(key, []).append(word)
    return list(groups.values())
```
