Arrays are the foundation of most data structure problems. Understanding how to efficiently traverse, sort, and manipulate sequences directly unlocks solutions to many advanced problems.

## Key techniques

| Technique | Key idea | Time |
|-----------|----------|------|
| Prefix Sum | Precompute cumulative sums for O(1) range queries | O(n) build, O(1) query |
| Kadane's | Track max subarray ending here vs reset to 0 | O(n) |
| Merge Intervals | Sort by start, merge overlapping | O(n log n) |
| Matrix Rotation | Transpose then reverse rows | O(n²) |

## Representative approach

```python
def prefix_sum(arr):
    """Compute prefix sums for fast range queries."""
    prefix = [0]
    for num in arr:
        prefix.append(prefix[-1] + num)
    # Query: sum(arr[i:j+1]) = prefix[j+1] - prefix[i]
    return prefix

def kadane(nums):
    """Find maximum subarray sum in a single pass."""
    max_ending_here = max_so_far = nums[0]
    for num in nums[1:]:
        max_ending_here = max(num, max_ending_here + num)
        max_so_far = max(max_so_far, max_ending_here)
    return max_so_far
```
